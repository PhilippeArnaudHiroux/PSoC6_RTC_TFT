# PSoC6_RTC_TFT
In this project a clock is made that you can set using the capsense sensor.

## What has been used
### Meterial
* PSoC 6 wifi-bt pioneer kit
* CY8CKIT-028-TFT display shield

### Software
* ModusToolbox

### Peripheries
* Capsense sensor (slider, button 0 and button 1)
* TFT-display
* Ambient light sensor
* RTC
* Timer
